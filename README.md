# Smart Expense Tracker v1

A beginner Python CLI app I built in Week 1 of learning Backend.

**What it does:**
- Asks for 3 expenses using `input()`
- Stores them in a `list` 
- Calculates total + average with a `for` loop
- Warns you if total > ₦2000 using `if/else`

**Tech Used:** Python, Lists, Loops, Conditionals
